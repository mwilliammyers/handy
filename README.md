# handy

A collection of [fishshell] functions I find handy.

## Prerequisites
- [`rsync`]
- [`eza`]

## Install

With [`fisher`]:

```
fisher install mwilliammyers/handy
```

[`fish`]: https://fishshell.com
[`rsync`]: https://rsync.samba.org/
[`eza`]: https://the.exa.website/
